> `Vue` administrator dashboard, implemented by `@vuejs` (including `vue-router`, `vuex`)

### Usage

```shell
git clone https://github.com/lbwa/adminize.git --depth=1
cd adminize
yarn serve
# visit http://localhost:8080 (or http://<your host device ip>:8080)
```
